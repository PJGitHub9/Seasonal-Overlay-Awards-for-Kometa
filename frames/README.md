# Gold & Silver Frame Overlays

These PNG files should be created as semi-transparent frames that overlay on movie/TV posters.

## Specifications:

### Dimensions
- Width: 2000px (will scale to poster width)
- Height: 3000px (will scale to poster height)
- Format: PNG with transparency

### Design

**Gold Frames (Winners):**
- **Color**: Gold (#FFD700 or similar metallic gold)
- **Style**: Similar to UMTK's red frame but in gold
- **Border**: Thick decorative border around the edges
- **Transparency**: Center should be completely transparent to show the poster

**Silver Frames (Nominees):**
- **Color**: Silver (#C0C0C0 or similar metallic silver)
- **Style**: Same as gold but in silver
- **Border**: Same thickness as gold frames
- **Transparency**: Center should be completely transparent

### Files Needed

1. **gold_frame_top.png** - Winners with thicker border at TOP
2. **gold_frame_bottom.png** - Winners with thicker border at BOTTOM (default)
3. **silver_frame_top.png** - Nominees with thicker border at TOP
4. **silver_frame_bottom.png** - Nominees with thicker border at BOTTOM (default)

## Creating the Frames

You can create these using:
1. **Photoshop/GIMP**: Create a 2000x3000 canvas, add gold borders
2. **Online tools**: Use Canva or similar tools
3. **Reference UMTK**: Check the UMTK red frame files for style inspiration
